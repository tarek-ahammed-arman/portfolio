# Tarek Ahammed Arman — Personal Portfolio

Welcome to my cybersecurity-focused portfolio. This site showcases my hands-on experience, technical skills, labs, projects, research, and certifications, providing a clear view of my expertise in cybersecurity.

## Portfolio Focus

**This portfolio emphasizes my cybersecurity journey and achievements:**

- **Cybersecurity Labs & Learning Paths:** Hands-on labs covering networking, Linux, Windows, web security, cryptography, incident response, and SIEM.
- **Ethical Hacking & Penetration Testing:** Practical exercises on vulnerability assessment, exploitation, and secure system design.
- **Full-Stack Web Development:** Projects with live demos and source code
- **Research & Publications:** Academic contributions and blockchain studies
- **Certifications:** Professional courses and training
- **Skills:** Technical proficiencies across programming, web, databases, and security

This portfolio also includes projects and labs demonstrating applied cybersecurity techniques and problem-solving in real-world scenarios.

## Technologies Used

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS, shadcn/ui
- **Icons & Animations:** Lucide-react, Tailwind transitions
- **Forms & Validation:** react-hook-form, zod
- **Email Handling:** EmailJS

## Getting Started

**Follow these steps to run the project locally:**

1. Clone the repository

    ```bash
    git clone https://github.com/tarek-ahammed-arman/portfolio.git

    cd portfolio
2. Install dependencies

    ```bash
    npm install
3. Run the development server
    ```bash
     npm run dev
4. Open the site

The development server will start, and the site will be available at:http://localhost:8080/

## Project Structure

- public/ → Static files (favicon, images, etc.)
- src/components/ → Reusable UI components (Header, Hero, ExpertiseAndCredentials EducationAndResearch, LabsAndProjects, Contact)
- src/pages/ → Page-level components
- src/assets/ → Images for projects, labs, and hero sections
